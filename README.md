# Live Stock Market Dashboard 📈

Real-time stock dashboard with technical indicators,
performance analytics and multi-stock comparison.

## Live Demo
[Click here](https://stock-dashboard-webapp-fqkgwqn7fq58kq6ee4xappd.streamlit.app)

## Features
- Live price chart with 20 & 50 day moving averages
- Volume bars colored green/red by day direction
- Monthly returns visualization
- Sharpe ratio, volatility, max drawdown stats
- Compare up to 6 stocks side by side
- Includes Indian stocks (NSE) support

## Tools Used
- Python, Streamlit, yfinance, Matplotlib, Pandas

## How to Run Locally
pip install streamlit yfinance pandas numpy matplotlib
streamlit run app.py
